# Node.js Dynamic Routing Handler
Mengembangkan aplikasi berbasis Node.js yang berfokus pada efisiensi manajemen jalur (routing) pada server. Proyek ini menunjukkan kemampuan saya dalam membangun fondasi aplikasi web yang skalabel dengan menangani berbagai permintaan HTTP secara dinamis.

Proyek ini mendemonstrasikan cara menangani rute (routing) secara dinamis di lingkungan Node.js tanpa menggunakan framework tambahan (Vanilla Node.js).

## Fitur
- Pengaturan rute otomatis berdasarkan parameter URL.
- Penanganan respons server untuk berbagai path.
- Struktur kode modular pada `server.js`.

## Cara Menjalankan
1. Pastikan Node.js sudah terinstal di komputer Anda.
2. Clone repositori ini.
3. Jalankan perintah berikut di terminal:
   node server.js
4. Akses server melalui browser (di localhost:3000 atau port yang Anda tentukan).
